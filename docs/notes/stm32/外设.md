# STM32 外设与常见英文缩写

STM32 常用外设在资料中经常以英文缩写出现，理解这些缩写有助于读懂参考手册、例程和寄存器配置。

## 常见外设缩写

- `NVIC` (Nested Vectored Interrupt Controller)：嵌套向量中断控制器
- `SysTick` (System Tick Timer)：系统滴答定时器
- `RCC` (Reset and Clock Control)：复位和时钟控制
- `GPIO` (General Purpose Input/Output)：通用 IO 口
- `AFIO` (Alternate Function Input/Output)：复用 IO 口
- `EXTI` (External Interrupt)：外部中断
- `TIM` (Timer)：定时器
- `ADC` (Analog to Digital Converter)：模数转换器
- `DMA` (Direct Memory Access)：直接内存访问
- `USART` (Universal Synchronous/Asynchronous Receiver/Transmitter)：同步/异步串口通信
- `I2C` (Inter-Integrated Circuit)：I2C 通信
- `SPI` (Serial Peripheral Interface)：SPI 通信
- `CAN` (Controller Area Network)：CAN 通信
- `USB` (Universal Serial Bus)：USB 通信
- `RTC` (Real Time Clock)：实时时钟
- `CRC` (Cyclic Redundancy Check)：CRC 校验
- `PWR` (Power)：电源控制
- `BKP` (Backup Registers)：备份寄存器
- `IWDG` (Independent Watchdog)：独立看门狗
- `WWDG` (Window Watchdog)：窗口看门狗
- `DAC` (Digital to Analog Converter)：数模转换器
- `SDIO` (SD Input/Output)：SD 卡接口
- `FSMC` (Flexible Static Memory Controller)：可变静态存储控制器
- `USB OTG` (USB On-The-Go)：USB 主机接口

## 这些外设的用途

- `NVIC`：负责中断嵌套优先级管理和中断使能，是 Cortex-M 内核中断体系的核心。
- `SysTick`：用于系统定时、任务切换、操作系统节拍等基础计时需求。
- `RCC`：配置系统时钟源、PLL、总线时钟，并控制外设时钟的开关。
- `GPIO`：处理单片机与外部数字信号的输入/输出，也是大多数外设复用口的基础。
- `AFIO`：在 STM32F1 这类器件上，用于选择引脚复用功能，例如串口、定时器 PWM、外部中断等功能对应的引脚映射。
- `EXTI`：将外部引脚信号转换为中断请求，适用于按键、外部传感器触发等场景。
- `TIM`：定时器模块用于定时、计数、PWM 输出、输入捕获、编码器接口等多种功能。
- `ADC`：将模拟电压转换成数字量，常用于传感器采集、按键电压检测等。
- `DMA`：在外设与内存之间传输数据时可减轻 CPU 负担，提高系统效率。
- `USART`：串口通信模块，支持同步和异步模式，用于串口调试、外设通信、Modbus 等。
- `I2C`：常用于连接温度传感器、EEPROM、RTC 等外设的两线串行总线。
- `SPI`：高速串行外设接口，适合连接 Flash、LCD、传感器等设备。
- `CAN`：用于车载或工业网络通信，支持多节点、差分传输。
- `USB`：用于与 PC 或 USB 设备通信，常见于调试、数据传输等。
- `RTC`：实时时钟模块可用于保持日历时间、闹钟、唤醒功能。
- `CRC`：循环冗余校验，常用于数据完整性校验。
- `PWR`：电源和低功耗管理，涉及睡眠、待机、备份域等。
- `BKP`：备份寄存器域，用于备份存储数据，如 RTC 设置或唤醒标志。
- `IWDG`：独立看门狗，可在程序卡死时复位系统，增强可靠性。
- `WWDG`：窗口看门狗，比独立看门狗增加了窗口期限制。
- `DAC`：将数字值转换为模拟电压，常用于音频、模拟控制信号输出。
- `SDIO`：用于与 SD 卡或 MMC 存储卡通信。
- `FSMC`：灵活静态存储控制器，用于访问 NOR/NAND Flash、SRAM、LCD 控制器等外部存储设备。
- `USB OTG`：在 USB 主机和设备模式之间切换，支持即插即用。
