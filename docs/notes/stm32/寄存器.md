| 寄存器名称（中文）         | 寄存器名称（英文）                                                       | 作用说明                               |
| ----------------- | --------------------------------------------------------------- | ---------------------------------- |
| **复位和时钟控制寄存器**    | RCC (Reset and Clock Control)                                   | 用于管理复位、时钟源配置、外设时钟等。                |
| **时钟源控制寄存器**      | RCC_CR (Clock Control Register)                                 | 配置时钟源，如主时钟、PLL等。                   |
| **时钟配置寄存器**       | RCC_CFGR (Clock Configuration Register)                         | 配置时钟源、分频器和系统时钟。                    |
| **GPIO端口配置寄存器**   | GPIOx_MODER (GPIO Mode Register)                                | 配置 GPIO 引脚的工作模式（输入、输出、复用功能、模拟模式等）。 |
| **GPIO端口输出数据寄存器** | GPIOx_ODR (GPIO Output Data Register)                           | 控制 GPIO 引脚的输出电平（高电平或低电平）。          |
| **GPIO端口输入数据寄存器** | GPIOx_IDR (GPIO Input Data Register)                            | 读取 GPIO 引脚的输入电平（高电平或低电平）。          |
| **GPIO端口拉电阻寄存器**  | GPIOx_PUPDR (GPIO Pull-Up/Pull-Down Register)                   | 配置 GPIO 引脚的上拉或下拉电阻。                |
| **GPIO端口输出类型寄存器** | GPIOx_OTYPER (GPIO Output Type Register)                        | 配置 GPIO 引脚的输出类型（推挽或开漏）。            |
| **外部中断线控制寄存器**    | EXTI (External Interrupt)                                       | 配置外部中断线的触发方式（上升沿、下降沿或双边沿），以及中断使能。  |
| **外部中断线触发寄存器**    | EXTI_IMR (Interrupt Mask Register)                              | 配置每个外部中断线路的中断使能。                   |
| **中断控制寄存器**       | NVIC (Nested Vectored Interrupt Controller)                     | 配置中断的优先级和使能中断。                     |
| **定时器控制寄存器**      | TIMx_CR1 (Timer Control Register)                               | 配置定时器的计数方向、时钟源、预分频系数等参数。           |
| **定时器自动重载寄存器**    | TIMx_ARR (Timer Auto-Reload Register)                           | 配置定时器的自动重载值，控制计时周期。                |
| **定时器捕获/比较寄存器**   | TIMx_CCR (Timer Capture/Compare Register)                       | 用于定时器的输入捕获或输出比较功能。                 |
| **定时器死区时间寄存器**    | TIMx_BDTR (Timer Break & Dead-Time Register)                    | 配置定时器的死区时间，通常用于PWM信号控制。            |
| **定时器重复计数寄存器**    | TIMx_RCR (Timer Repetition Counter Register)                    | 配置定时器的重复计数，用于循环模式。                 |
| **DMA控制寄存器**      | DMA (Direct Memory Access)                                      | 配置 DMA 模块，用于内存和外设之间的数据传输。          |
| **DMA传输方向寄存器**    | DMA_CCR (DMA Channel Control Register)                          | 配置 DMA 渠道的传输方向、传输大小、使能等。           |
| **闪存控制寄存器**       | FLASH (Flash Memory)                                            | 配置闪存的读写操作，如页擦除、编程等。                |
| **实时时钟控制寄存器**     | RTC (Real-Time Clock)                                           | 配置 RTC 模块，进行时间和日期的管理。              |
| **电源控制寄存器**       | PWR (Power Control)                                             | 配置芯片的电源管理，包括待机模式、睡眠模式等。            |
| **看门狗控制寄存器**      | IWDG (Independent Watchdog)                                     | 配置独立看门狗的计时和复位功能，用于防止系统死锁。          |
| **I2C控制寄存器**      | I2C (Inter-Integrated Circuit)                                  | 配置 I2C 总线的主从模式、传输速度、时钟等。           |
| **SPI控制寄存器**      | SPI (Serial Peripheral Interface)                               | 配置 SPI 总线的工作模式、时钟极性等。              |
| **USART控制寄存器**    | USART (Universal Synchronous Asynchronous Receiver Transmitter) | 配置串口通信的波特率、数据位、校验等参数。              |
| **USB控制寄存器**      | USB (Universal Serial Bus)                                      | 配置 USB 总线的工作模式，如主机、设备、传输速率等。       |
| **CRC控制寄存器**      | CRC (Cyclic Redundancy Check)                                   | 配置 CRC 校验模块，用于数据完整性检测。             |
| **看门狗时间寄存器**      | WWDG (Window Watchdog)                                          | 配置窗口看门狗，用于监控系统，防止系统进入死锁。           |
| **数据缓存控制寄存器**     | DCache (Data Cache Control)                                     | 配置数据缓存，控制数据存储的缓存策略。                |
| **专用功能寄存器**       | AFIO (Alternate Function I/O)                                   | 配置 I/O 引脚的替代功能。                    |
| **故障检测寄存器**       | FDCAN (Fault Detection CAN)                                     | 配置 CAN 总线的故障检测功能。                  |
| **死区时间控制寄存器**     | DTG (Dead Time Generator)                                       | 配置定时器的死区时间，通常用于电机控制等场景中的 PWM 波形控制。 |
| **重复计数寄存器**       | REP (Repetition Counter)                                        | 配置 PWM 模式下的重复计数值，用于控制周期性信号的重复次数。   |
