## 💡 答题模板速查

### 通用回答结构
```
1. 【定义理解】什么是XXX
2. 【工作原理】怎样工作/流程图
3. 【应用场景】在哪用/为什么
4. 【权衡对比】vs类似方案的区别
5. 【实战细节】项目中怎么做/踩过什么坑
```

### 协议/通信题答题模板
```
1. 物理层：几条线、电平标准
2. 协议层：帧格式、通信速率
3. 特点：全双工/单工、同步/异步
4. 应用：典型应用场景
5. 问题：常见问题和解决方案
```

### 算法/原理题答题模板
```
1. 核心算法伪代码/流程图
2. 复杂度分析（时间/空间）
3. 实现细节和注意事项
4. 优化方向
5. 项目应用案例
```
