# Arduino / AVR 基础

## 1. AVR 端口寄存器

在 AVR 里，一个端口（比如 `PORTB`）通常会配套三类寄存器：

- `DDRB`：方向寄存器，`1 = 输出`，`0 = 输入`
- `PORTB`：输出寄存器
  输出模式下写 `1/0` 就是高/低电平；输入模式下写 `1` 会开启上拉
- `PINB`：读取引脚电平
  某些芯片里也常用于“写 1 翻转”特性，但刚开始先不要混用

补充：

- AVR C 里无限循环一般写成 `while (1) { ... }`
- `while (1)` 这类基础定义通常来自 `<avr/io.h>`
- `_delay_ms(x)` 来自 `<util/delay.h>`，单位是毫秒

## 2. 置高 / 置低

```c
PORTB |= (1 << 7);   // 置高
PORTB &= ~(1 << 7);  // 置低
```

## 3. Timer 的基本理解

Timer 可以先理解成：

> 一个会自动加 1 的计数器 + 一组控制它如何计数的寄存器

- `TCCR2A`、`TCCR2B`：控制寄存器，决定 `Timer2` 怎么计数
- `TCNT2`：当前计数寄存器，表示现在数到几

例如 `TCNT2 = 6`，而 `Timer2` 是 8-bit（`0 ~ 255`），它就会：

```text
6 -> 7 -> 8 -> ... -> 255 -> 0
```

从 `255` 再加 1 回到 `0` 的过程叫“溢出”。发生溢出时，硬件会把 `TOV2 = 1`。

## 4. 状态标志寄存器

- `TIFR2`：`Timer2` 的状态 / 标志寄存器
- `TOV2`：溢出标志

可以把 `TIFR2` 理解成 Timer2 的“状态指示灯面板”，里面每一位都表示某件事有没有发生。

## 5. Clock Select（时钟选择）

`CS22 / CS21 / CS20` 用来决定 Timer 数得有多快。

| `CS22` | `CS21` | `CS20` | 含义 |
| ------ | ------ | ------ | ---- |
| 0 | 0 | 0 | 不数（停） |
| 0 | 0 | 1 | 不分频（16 MHz） |
| 0 | 1 | 0 | `/8` |
| 0 | 1 | 1 | `/32` |
| 1 | 0 | 0 | `/64` |
| 1 | 1 | 0 | `/256` |
| 1 | 1 | 1 | `/1024` |
