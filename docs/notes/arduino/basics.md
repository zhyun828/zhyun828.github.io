1.  在 AVR 里，一个端口（比如 PORTB）通常有三类寄存器：

    DDRB：方向寄存器，1=输出，0=输入

    PORTB：输出寄存器（当是输出时写 1/0 就是高/低电平；当是输入时写 1 会开上拉）

    PINB：读引脚电平（也常用于某些“写 1 翻转”特性，但先别混用）

    AVR C 里无限循环就是：while (1) { ... } 来自 <avr/io.h>

    _delay_ms(x) 来自 <util/delay.h>，单位 ms

2.  置高/置低：

    高：PORTB |= (1<<7);
    低：PORTB &= ~(1<<7);

3.  Timer = 一个会自动加 1 的计数器 + 一堆控制它的“开关寄存器

    控制寄存器 决定 Timer 怎么数  `TCCR2A`, `TCCR2B` 

    计数寄存器 现在数到几了       `TCNT2`
    例如 TCNT2 = 6; Timer2 是 8-bit（0 ~ 255） 它会6 → 7 → 8 → ... → 255 → 0（溢出）
    溢出是什么意思？当它从 255 再 +1 回到 0，硬件会：把 TOV2 = 1
       
    状态/标志  有没有“出事”（溢出）   `TIFR2`（里的 `TOV2`） 
    TIFR2（Timer Interrupt Flag Register） —— Timer2 的“状态指示灯面板” 它里面每一位表示：某件事有没有发生过
    TOV2 —— 溢出标志

    CS22	CS21	CS20    Clock Select 时钟选择 决定 Timer 数得多快
    | CS22  | CS21  | CS20  |  含义          |
    | ----- | ----- | ----- | -------------- |
    | 0     | 0     | 0     | 不数（停）      |
    | 0     | 0     | 1     | 不分频（16 MHz）|
    | 0     | 1     | 0     | /8             |
    | 0     | 1     | 1     | /32            |
    | 1     | 0     | 0     | /64            |
    | 1     | 1     | 0     | /256           |
    | 1     | 1     | 1     | /1024          |
